# 발주서 처리 프로그램

사방넷 발주서를 거래처별로 자동 분리하고 출고리스트를 생성하는 Windows 프로그램입니다.

## 주요 기능

- **거래처별 발주서 자동 분리** - 아이디 → 거래처 매핑으로 엑셀 파일 분리
- **합포 자동 감지** - 파란색 셀(FF66CCFF) 자동 인식
- **해외배송/훼손 구분 처리** - 배송메세지/수집옵션명으로 자동 판별
- **출고리스트 자동 생성** - A4 가로 인쇄 최적화 (한 장 40행)
- **구글시트 연동** - 위치 파일 자동 다운로드
- **자동복구** - 손상된 발주서 파일 복구 시도
- **재고 자동 계산** - 전체 출고분 차감한 잔여재고 표시
- **자동 업데이트 알림** - 새 버전 출시 시 알림

## 다운로드

최신 버전: **[Releases 페이지](../../releases/latest)**

1. `발주서처리프로그램.exe` 다운로드
2. 바로 실행 (설치 불필요)

## 사용 방법

### STEP 1: 파일 선택
- ① 위치 파일 (구글시트 자동 다운로드 지원)
- ② 발주서 파일 (사방넷에서 다운로드)
- ③ 재고 파일 (재고현황조회)
- ④ 저장 폴더

### STEP 2: 매핑 확인
- 아이디 ↔ 거래처명 매핑 설정/수정

### STEP 3: 미리보기
- 생성될 파일 목록 확인

### STEP 4: 처리 실행
- 거래처별 xlsx 파일 자동 생성

### STEP 5: 출고리스트 생성
- A4 인쇄 최적화된 출고리스트

## 개발자용

### 로컬 빌드
```bash
pip install pyinstaller openpyxl pandas tkinterdnd2
pyinstaller --onefile --windowed --name "발주서처리프로그램" --add-data "version.txt;." main.py
```

### 릴리스 방법
```bash
# 1. version.txt 수정
echo "1.0.1" > version.txt

# 2. 커밋 & 태그 푸시
git add version.txt main.py
git commit -m "v1.0.1 업데이트"
git tag v1.0.1
git push origin main --tags

# 3. GitHub Actions가 자동으로 빌드 및 Release 생성
```
